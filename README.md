### Usage

1. Download your favourite accent from the `themes/` folder.
1. Move the file to `~/.config/zed/themes`.
1. Open the command palette (<kbd>Ctrl+Shift+p</kbd>) in zed, use `theme
   selector: toggle` and select your variant.
